# DockerPractices
Docker Practices

## 1.1. Docker Commands

### 1.1.1. Build Command

> -t: tag
> hello-docker: the tag/name

```console
docker build -t hello-docker
```

### 1.1.2. Check Image List

```console
docker image ls
```

### 1.1.3. Run Docker Image

```console
docker run hello-docker
```
